== Description ==

Our simple and powerfull wordpress content protection plugin will help you to detect all changes on your posts and pages. It has built in calendar so you can restore all possible changes on your website in a few clicks of the button. 
You don't have to make a full backup of your website when you decide to make some changes in your articles, with our free contet protection plugin you can backup just a pages you need to restore. This is a great tool for SEO as well, you can do try to make some changes on your website pages and if Google doesn't like it you can just restore your website content to the date you made changes on.



== Installation ==

1. Upload all the files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
